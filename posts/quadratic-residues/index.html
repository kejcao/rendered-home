Quadractic Residues

Jul 2, 2024

I recently learned about quadratic residues, a concept from number theory. A quadratic residue is an integer that is congruent to a perfect square modulo $n$. In other words, an integer $x$ modulo $n$ is a quadratic residue if and only if there exists another integer $a$, such that,

$$ a^2 \equiv x \pmod{n}. $$

This relationship implies that $a$ can be thought of as a "square root." Furthermore, whenever $a$ is a square root of $x$, it's also true that $-a$ is a square root of $x$.

The two values $a$ and $-a$ modulo $n$ aren't necessarily distinct, for example when $a=2$ and $n=8$. However, if the modulo $n$ is odd, then the solutions $a$ and $-a$ are always distinct because $x\not\equiv -x \pmod{n}$ for all $x$ if and only if $n$ is odd.

Let $p$ be an odd prime and $g$ a generator of $(\mathbb{Z} / p\mathbb{Z})^{\times}$. Recall that this group is cyclic and has order $p-1$, with elements ranging from 1 to $p-1$ inclusive. Every element $a\in(\mathbb{Z} / p\mathbb{Z})^{\times}$ is expressible as a power of the generator $g^k=a$ such that $0 \leq k \leq p-2$. Now, consider an element $a=g^{2k}$ where the exponent of the generator is even. $a$ is a quadratic residue because $g^{2k} = (g^k)^2$. Hence, at least half of the elements in $(\mathbb{Z} / p\mathbb{Z})^{\times}$ are quadratic residues.

Furthermore, because each quadratic residue has two unique square roots ($x\neq -x$ since $p$ is odd) there must be at most $(p-1)/2$ quadratic residues—one more, and that would imply $(\mathbb{Z} / p\mathbb{Z})^{\times}$ has order greater than $p-1$. Therefore, exactly half of the elements in the multiplicative group of integers modulo $p$ are quadratic residues.

So, only elements of the form $g^{2k}$ are quadratic residues. Can we exploit this fact to easily test whether a value is a quadratic residue? Given an element $a=g^n$ consider the result of $a^{(p-1)/2}$. If $n=2k$, then,

$$a^{(p-1)/2} = (g^{2k})^{(p-1)/2} = g^{k(p-1)} = 1.$$

Therefore, if $a$ is a quadractic residue, then the result of $a^{(p-1)/2}$ is always 1. What if $a$ is a quadratic non-residue—in other words, $n=2k+1$?

$$\begin{aligned} a^{(p-1)/2} &= (g^{2k+1})^{(p-1)/2} \\ &= g^{(2k+1)(p-1)/2} \\ &= g^{k(p-1)}g^{(p-1)/2} \\ &= g^{(p-1)/2} \end{aligned}$$

and because $(g^{(p-1)/2})^2 = 1$, the value of $g^{(p-1)/2}$ is either 1 or -1. Since the exponent is not a multiple of $(p-1)$, the value cannot be 1 and therefore must be -1. So, we have shown that $a^{(p-1)/2}$ is equal to 1 if $a$ is a quadratic residue and $-1$ if $a$ is a quadractic non-residue. This fact is called Euler's Criterion. We denote this with the Legendre symbol,

$$\left(\frac{a}{p}\right) = a^{(p-1)/2}.$$

This is also written as $(a\mid p)$ or $(a/p)$.
