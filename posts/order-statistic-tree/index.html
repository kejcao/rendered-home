DD: An Order Statistic Tree in Haskell

Nov 17, 2024

To practice my Haskell, I implemented a basic order statistic tree. Let's first construct the datatype and implement the Show typeclass with something a little easier to visualize.

data Tree a
  = Empty
  | Node Int (Tree a) a (Tree a)
  deriving (Eq, Foldable)

push :: String -> String
push = unlines . map ("      " ++) . lines

instance (Show a) => Show (Tree a) where
  show Empty = ""
  show (Node s l v r) =
    concat
      [ (push . show) l,
        "\n",
        show v ++ "(" ++ show s ++ ")",
        "\n",
        (push . show) r
      ]

It is now easy to create a function to insert an element into the tree while obeying the size invariant.

insert :: (Ord a) => a -> Tree a -> Tree a
insert x Empty = Node 1 Empty x Empty
insert x (Node s l v r) = case compare x v of
  EQ -> Node s l v r
  LT -> Node (s + 1) (insert x l) v r
  GT -> Node (s + 1) l v (insert x r)

We can construct a random tree and print it.

randomTree :: IO (Tree Int)
randomTree = do
  l <- replicateM 128 (randomRIO (1, 128 :: Int))
  return $ foldl (flip insert) Empty (nub l)

main :: IO ()
main = randomTree >>= print

Deletion is a little harder to implement.

delete :: (Ord a) => a -> Tree a -> Either (Tree a) (Tree a)
delete _ Empty = Left Empty
delete x (Node s Empty v Empty) =
  if x == v
    then Right Empty
    else Left (Node s Empty v Empty)
