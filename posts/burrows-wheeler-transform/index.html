Burrows–Wheeler Transform

Jun 28, 2024

I've known about the Burrows–Wheeler transform for a while but have never really understood how it worked. 

import Data.List (sort)

bwtMatrix s = sort rotations
  where
    rotations = scanl
      (\(c : s) _ -> s ++ [c])
      s [1 .. length s - 1]

Here is an example

> bwtMatrix "cocoon$"
["$cocoon","cocoon$","coon$co","n$cocoo","ocoon$c","on$coco","oon$coc"]

If we have a string $s_0s_1\dotsc s_n$ note how in the left rotation $s_1\dotsc s_ns_0$ we have only moved the letter $s_0$ from the left to the right. The substring $s_1\dotsc s_n$ is present, unchanged, in both strings.

In a BWT matrix, the position of the $n$th occurrence of a letter in the first column in the original string is equal to the position of the $n$th occurrence of a letter in the last column in the original string. In the example below, the first "o" in the first column is in the second position of the original string, the second "o" is in the fifth position of the original string, and the third "o" is in the fourth position of the original string. 

$cocoon    $cocoon
cocoon$    cocoon$
coon$co    coon$co
n$cocoo    n$cocoo
ocoon$c    ocoon$c
on$coco    on$coco
oon$coc    oon$coc
