DD: Globe

Nov 15, 2024

We can describe a sphere centered around the origin with a radius of one by the equation $\mathbf{x}^2 = 1$. Any point $\mathbf{x} \in \mathbb{R}^3$ that satisfies the equation is a point on the sphere. Note that $\mathbf{x}^2$ refers to the dot product. We can substitute $\mathbf{x}$ with the equation for a ray $r(t) = \mathbf{o} + t\mathbf{d}$ and solve for $t$,

$$\begin{aligned} (\mathbf{o} + t\mathbf{d})^2 &= 1 \\ \mathbf{o}^2 + 2t\mathbf{d} + (t\mathbf{d})^2 &= 1 \\ \mathbf{d}^2t^2 + (2\mathbf{o}\mathbf{d})t + \mathbf{o}^2 - 1 &= 0 \end{aligned}$$

And the final quadratic can be solved easily to find the intersection between the ray with the sphere. In code,

def intersect(o, d):
    a = d.dot(d)
    b = 2 * o.dot(d)
    c = o.dot(o) - 1

    det = b**2 - 4*a*c

    if det < 0:
        return None
    else:
        t = min(
            (-b + np.sqrt(det)) / (2*a),
            (-b - np.sqrt(det)) / (2*a),
        )
        return o + t*d
